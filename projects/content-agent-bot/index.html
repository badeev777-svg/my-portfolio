Архитектура

Мультиагентная система

Бот использует несколько специализированных агентов для решения разных задач

📄 Scraper
Извлекает тезисы из URL с помощью httpx + BeautifulSoup4. Результат передаётся дальше.
🎯 Strategist
Строит 7-дневную воронку продаж: engagement → expertise → trust → sale. Планирует посты по психологии.
✍️ Copywriter
Генерирует посты для Telegram, VK, Stories. Каждая платформа — отдельный промпт и стиль.
💰 Monetization
3 бесплатных генерации + лимит на переписи (3/день для платных). Хранит метрики использования.
🔐 Telegram Payments
Интегрированная оплата через BotFather. Автоматическая активация подписки.

Управление

Команды бота

/start
Онбординг новых пользователей. Настройка профиля стиля (тон, целевая аудитория, формат).
/style
Показать текущий профиль стиля. /style_edit — изменить параметры.
[URL]
Отправить ссылку на статью → бот анализирует и генерирует контент-план на 7 дней.
/examples
Добавить примеры своих постов. Бот будет анализировать их для улучшения стиля.
/history
Показать последние 5 генераций с возможностью воспроизведения.
/subscribe
Перейти на оплату подписки (2499 ₽/мес). Безлимит генераций и 3 переписи/день.
