# Магазин на Flask

Простой интернет-магазин на Python с каталогом товаров и корзиной.

## Установка

1. Перейдите в папку проекта:
   ```bash
   cd shop
   ```
2. Установите зависимости:
   ```bash
   python -m pip install -r requirements.txt
   ```

## Запуск

```bash
python app.py
```

Откройте в браузере http://127.0.0.1:5000

## Что есть

- Каталог товаров
- Добавление в корзину
- Удаление товара
- Очистка корзины

> Если хочешь, могу добавить оплату, фильтр товаров или страницы товара.
