# Портфолио — Бадеев Александр

## Что это за проект

Одностраничный сайт-портфолио для Бадеева Александра — разработчика веб-приложений и AI-решений. Сайт показывает выполненные проекты потенциальным клиентам и помогает им принять решение о заказе. Содержит форму обратной связи, ссылки на демо-версии проектов и контакты.

## Для кого

Частные клиенты — малый бизнес и ИП: владельцы магазинов, риелторы, сервисные компании, предприниматели, которым нужен сайт, CRM или автоматизация.

## Страницы / разделы (текущие)

Сайт одностраничный. Разделы на странице `index.html`:

| Раздел | ID | Описание |
|--------|----|----------|
| Герой | `#hero` | Имя, должность, статистика, кнопки CTA |
| Обо мне | `#about` | Краткое описание, чипы навыков |
| Услуги | `#services` | 4 карточки: лендинги, CRM, нейросети, мобильные приложения |
| Проекты | `#projects` | 7 карточек с превью, описанием и ссылками на демо |
| Навыки | `#skills` | Прогресс-бары + облако технологий |
| Процесс | `#process` | 4 шага работы: бриф → прототип → разработка → сдача |
| Контакт | `#cta` | Форма обратной связи + кнопки Telegram, Email, GitHub, телефон |

## Страницы планируются

Новых страниц не планируется — сайт остаётся одностраничным.  
Планируется публикация на хостинг (дата не определена).

## Дизайн

- **Тема:** тёмная (dark mode)
- **Фон:** `#08090e` / `#0f1117`
- **Акценты:** градиент фиолетовый → циан (`#7c6cff` → `#00d4ff`)
- **Карточки:** `#141620` с тонкой фиолетовой рамкой
- **Шрифт:** Segoe UI, system-ui
- **Анимации:** fade-up при скролле, hover-эффекты на карточках, анимированные прогресс-бары
- **Адаптивность:** мобильное меню (бургер), responsive grid
